= Удаление варианта решения

Варианты решений можно удалять для всех этапов.
include::partial$excerpts.adoc[tags=apply]


.Чтобы удалить вариант решения:
. Откройте вкладку _Параметры этапа_. В области _Варианты решений_ карточки этапа выберите вариант решения, который требуется удалить.
. Нажмите на кнопку image:buttons/delete_red_x.png[Крест] *Удалить*, чтобы удалить вариант решения.
+
Изменения будут применены ко всем незапущенным этапам.
