= Добавить решение в лист согласования

Во время выполнении этапа согласования документа согласующие принимают xref:task-decisions.adoc[решение] по согласованию. По умолчанию список принятых решений копируется в _Лист согласования_.

[WARNING]
====
include::partial$excerpts.adoc[tags=multiple]
====

При необходимости можно отключить копирование решений.

.Чтобы отключить копирование решений этапа в "Лист согласования":
. Создайте новую или откройте существующую карточку этапа.
. Перейдите на вкладку xref:stage-task.adoc["Настройки заданий"].
. Установите флаг `*Не добавлять решение в лист согласования*`.
